 1 Systemanforderungen
========================================================================

1.1 Minimale Systemvoraussetzungen
----------------------------------

Betriebssystem:      Windows 98/Me/2000/XP

Prozessor:           Pentium III oder AMD Athlon 1,0 GHz Prozessor

Arbeitsspeicher:     128 MByte RAM

Festplattenspeicher: 6 GByte freier Festplattenspeicher

Grafik:              32 MB Windows kompatible Grafikkarte* 

Sound:               Windows kompatible Soundkarte. NVIDIA nForce oder 
                     andere Motherboards/Soundkarten mit Dolby Digital
                     Interactive Content Encoder benötigt für 
                     Dolby Digital Audio.*

DirectX:             DirectX Version 9.0b oder höher

                     Bei der Verwendung des Software-Renderers wird
                     DirectX nicht benötigt
                     
Modem:               33.6K Baud (für Spiel über Modem/Internet)

Spielen über Internet (TCP/IP) und LAN (TCP/IP) wird unterstützt.

1.2 Empfohlene Systemvoraussetzungen
------------------------------------

Betriebssystem:      Windows 98/Me/2000/XP

Prozessor:           Pentium IV oder AMD Athlon 2,0 GHz Prozessor

Arbeitsspeicher:     512 MByte RAM

Festplattenspeicher:   6 GByte freier Festplattenspeicher 
                 
Grafik:              NVIDIA GeForce 4 oder ATI Radeon Hardware T&L Karte mit
                     128 MB VRAM empfohlen

Sound:               Sound Blaster Audigy 2 ZS

DirectX:             DirectX Version 9.0b oder höher
                    
Modem:               Breitband-Internetverbindung

Spielen über Internet (TCP/IP) und LAN (TCP/IP) wird unterstützt.

* Die Hardware sollte kompatibel sein mit DirectX Version 9.0b oder
höher.

 2 Leistung
========================================================================

2.1 Bildauflösung
-----------------

Die Bildauflösung wird den größten Einfluss auf die Leistung des Spiels
haben, falls Sie nicht zu den Glücklichen gehören, die eine Grafikkarte
der neuesten Generation besitzen. Starten Sie das Spiel mit 16 Bit Farbtiefe
und einer Auflösung von 640x480 oder 512x384, um auch mit älterer Hardware
eine annehmbare Grafikleistung zu erreichen.

2.2 CPU-Geschwindigkeit
-----------------------

Die Leistung des Spiels hängt stark von der CPU-Geschwindigkeit, der
Speicher-Bandbreite und der Zwischenspeicherleistung ab. Das Spiel läuft daher
sehr viel besser mit Prozessoren der neuesten Generation, wie etwa denen von
AMD und Intel. Siehe dazu auch Abschnitt 1.2. 

2.3 Arbeitsspeicher
-------------------

Die Leistung von Unreal Tournament 2004 hängt auch stark vom verfügbaren
Arbeitsspeicher ab. Systeme mit wenig Arbeitsspeicher müssen häufiger zum Laden
von Daten auf die Festplatte zugreifen, was dazu führen kann, dass das Spiel
kurz hängt. Hat Ihr Rechner nur 128 MByte RAM (oder weniger), beenden Sie alle
nicht benötigten Programme, während Sie Unreal Tournament 2004 spielen.

2.4 Den Computer aufrüsten?
---------------------------

Falls Sie sich mit dem Gedanken tragen, Ihren Rechner aufzurüsten, hier einige
Tipps, die auf unseren Erfahrungen mit dem Betrieb von Unreal Tournament 2004
auf einer Vielzahl von verschiedenen Systemen beruhen:

  1 Den größten Leistungszugewinn bei Unreal Tournament 2004 erreichen Sie durch
    den Einbau einer schnellen Grafikkarte.

  2 Durch eine schnellere CPU erreichen Sie ebenfalls eine starke Verbesserung
    der Leistung von Unreal Tournament 2004.

  3 Auch eine große Menge an Arbeitsspeicher verbessert die Leistung. Das
    Aufrüsten Ihres Systems auf 512 MByte RAM sollte aufgrund der ständig
    fallende Preise eine erschwingliche Lösung darstellen.

2.5 Die Leistung verbessern
---------------------------

Unreal Tournament 2004 legt beim ersten Start, abhängig von Ihrer Hardware,
bestimmte Standardeinstellungen fest, wodurch eine gute Grafikleistung
garantiert sein sollte. Allerdings gibt es immer noch Möglichkeiten, die
Grafikleistung zu optimieren.

Besitzen Sie eine schnelle CPU, haben Auflösung und Texturdetail den größten
Einfluss auf die Grafikleistung. Sie können die Einstellungen also verändern,
bis Sie das gewünschte Verhältnis zwischen Bildqualität und Grafikleistung
erhalten.

Besitzen Sie eine schnelle Grafikkarte, aber eine eher langsame CPU, verringern
Sie das Welten-Detail unter 'Detail' im Menü 'Einstellungen'.

Sollten Sie sowohl eine schnelle CPU als auch eine schnelle Grafikkarte besitzen,
dann sollten Sie diesen Abschnitt eigentlich überspringen können :).

 3 Mögliche Probleme
========================================================================

3.1 Abstürzen des Spiels
------------------------

Wird Unreal Tournament 2004 mit der Nachricht "Kein virtueller Speicherplatz
verfügbar" beendet, sollten Sie Speicherplatz auf Ihrer Systemplatte (C:)
freimachen, um das Problem in Zukunft zu vermeiden. Da die großen Levels und
detailreichen Texturen in Unreal Tournament 2004 große Mengen an virtuellem
Arbeitsspeicher benötigen, sollten mindestens 500 MByte an freiem Speicher
auf Ihrer Systemplatte vorhanden sein, wenn Sie das Spiel starten. 

3.2 Sound
---------

Wenn Sie eine 3D Soundkarte wie z.B. die Sound Blaster Audigy Series
Soundkarte verwenden, können Sie unter 'Einstellungen/Audio' EAX über
den Menüpunkt 'Hardware 3D Sound + EAX' aktivieren.

Um eine gute 3D Klangqualität zu erhalten, müssen Sie die neueste Version
des Sound Blaster Audigy Treibers verwenden.

Verwenden Sie eine ältere Version des Treibers, KANN dies zu Soundproblemen
in Unreal Tournament 2004 führen (der Sound wird zwischen 30-50% langsamer
wiedergegeben). In diesem Fall sollten Sie wieder'Software 3D Audio' wählen.

Ist Ihr computer an ein 5.1 Lautsprecher-System angschlossen, aktivieren
Sie die Option 'Hardware 3D Audio', um sich vom 360 Grad Sound "umhauen"
zu lassen.

Es gibt auch noch einen abgesicherten Modus, falls Sie Probleme mit dem
Sound haben, wie etwa hängender Sound, Rauschen oder schlechte Tonqualität.

3.3 Netzwerk-Probleme
---------------------

Um mit einer akzeptablen Leistung über das Internet spielen zu können,
benötigen Sie für die Verbindung zu Ihrem Internet Service Provider
mindestens ein 33.6K Modem.

Bekannte Netzwerk-Probleme:

  * Wenn ein neuer Spieler einem Netzwerk-Spiel beitritt, kann es bei den
    anderen Clients zu einer Verzögerung von 1/4 Sekunde kommen, während
    die Spieldaten wie Raster, Oberflächen, etc. für den neuen Spieler
    geladen werden. Dies ist so beabsichtigt.

  * Die Leistung beim Spielen über das Internet hängt bei Unreal Tournament
    2004 sehr stark von der Bandbreite Ihrer Verbindung, der Latenz (Ping),
    und dem packet loss ab. Das Spiel ist bis zu einem Ping von 300 mSek.
    und 5% packet loss spielbar, und benötigt mindestens ein 33.6K Modem.
    Erreichen Sie mit Ihrer Verbindung diese Mindestvorgaben nicht, wird
    sich die Leistung des Spiels erheblich verringern.

3.4 Steuerung
-------------

Manche PC-Tastaturen können bestimmte Kombinationen von 3 oder mehr 
gleichzeitig gedrückten Tasten nicht erkennen.


3.5 Direct3D
------------

Vergewissern Sie sich, dass Sie die neuesten Treiber für Ihre Grafikkarte
verwenden, da Unreal Tournament 2004 die Hardware und Treiber weit stärker
beansprucht als die meisten anderen Spiele. Außerdem haben die Hardware-
Hersteller mit den neuesten Treibern einige Bugs behoben, die bei Unreal
Tournament 2004 zu Problemen führen können. Die neusten Treiber erhalten
Sie auf der Webseite Ihres Grafikkarten-Herstellers.

3.6 TNT2/ Kyro II/ Voodoo 3
---------------------------

Aufgrund fehlender Funktionen wie Textur-Kompression und Cubemap-Unterstützung
können TNT2, Kyro II, Voodoo 3, G400, ... Grafikkarten das Spiel nicht in
höchster Bildqualität darstellen. Da nicht komprimierte Texturen weitaus
größer sind als komprimierte, müssen Größe und Farbtiefe der Texturen
drastisch reduziert werden, damit sie von diesen Grafikkarten verarbeitet
werden können. Dies bedeutet, dass sich Bildgegenstände - vor allem in den
Menüs - teilweise überschneiden werden. 

Die fehlende Cubemap-Unterstützung bewirkt, dass Reflektionen im Spiel nicht
echt aussehen, auch wenn Sie während des Spiels kaum die Zeit haben werden,
dies zu bemerken.

3.7 OpenGL
----------

Der OpenGL-Renderer wird nicht offiziell von Windows unterstützt, er stellt
aber bei bestimmten Hardware/Treiber-Kombinationen eine gute Alternative dar,
da er bei einigen Treibern weniger Bugs hervorruft. Kommt es während des
Spielens zu keinen schweren Grafikfehlern, gibt es jedoch keinen Grund, den
OpenGL-Renderer zu verwenden. Im Gegensatz zu Unreal Tournament wurde Unreal
Tournament 2004 für D3D optimiert und bietet mit dem D3D-Renderer die beste
Bildqualität.

Der OpenGL-Renderer benötigt außerdem mehr Systemressourcen als der
D3D-Renderer. Der OpenGL-Renderer funktioniert bekanntermaßen nicht korrekt
mit ATI-Treibern vor Version 7.76.

3.8 DirectX 9.0b
----------------

DirectX 9.0b ist auf der Unreal Tournament 2004-CD enthalten. Es kann auch
unter der unten aufgeführten URL heruntergeladen werden, falls es nicht
bereits auf Ihrem System installiert ist.

   http://www.microsoft.com/windows/directx/default.aspx

3.9  Spiel startet im Fenster-Modus
-----------------------------------

Wenn das Spiel beim Starten keine direkte Verbindung zum Keyboard erhält,
startet es im Fenster-Modus. Klicken Sie in diesem Fall auf das Fenster
und drücken Sie ALT-ENTER, um den Vollbild-Modus zu aktivieren.

3.10 Software-Rendering
-----------------------

Sie können das Spiel sinnvolle Standardeinstellungen für das Software-
Rendering festlegen lassen, indem Sie es mit dem Anhang '-software' in der
Befehlszeile starten. Außerdem wird so Software-Rendering als Render-
Methode festgelegt, was hilfreich sein kann, wenn Sie Probleme mit Ihrem
Grafikkarten-Treiber haben oder keine 3D-Beschleunigerkarte besitzen.

   ut2004 -software

3.11 Joystick-Unterstützung
---------------------------

Unreal Tournament 2004 unterstützt Joysticks und Gamepads. Die Tasten
werden ebenso konfiguriert wie die Tasten der Tastatur, über das Menü
'Einstellungen', 'Eingabe' und 'Steuerung konfigurieren'.

Möchten Sie die Einstellungen für die Analogsticks verändern, müssen Sie
einige Änderungen in der Datei User.ini vornehmen (sie befindet sich
meist im Verzeichnis c:\UT2004\System, oder aber im Unterverzeichnis
'System' des Verzeichnisses, unter dem Sie das Spiel installiert haben).

Im Abschnitt [Engine.Input] der Datei User.ini finden Sie folgende
Einträge:
JoyX=Axis aStrafe SpeedBase=300.0 DeadZone=0.1
JoyY=Axis aBaseY SpeedBase=300.0 DeadZone=0.1 Invert=-1
JoyZ=
JoyR=
JoyU=
JoyV=Axis aBaseX SpeedBase=2.0 DeadZone=0.4
JoySlider1=Axis aLookUp SpeedBase=2.0 DeadZone=0.4
JoySlider2=

Es ist möglich, für den SpaceFighter andere Einstellungen
festzulegen, indem Sie die folgenden Aliasse anlegen:

SpaceFighter_JoyX
SpaceFighter_JoyY
SpaceFighter_JoyZ
SpaceFighter_JoyR
SpaceFighter_JoyU
SpaceFighter_JoyV
SpaceFighter_JoySlider1
SpaceFighter_JoySlider2

Dies sind die Standard-Einstellungen für die Steuerung
des SpaceFighters:

// MouseX zugewiesen an JoyX
Aliases[32]=(Command="Axis aBaseX SpeedBase=100.0 DeadZone=0.1",
Alias="SpaceFighter_JoyX")

// MouseY zugewiesen an JoyY
Aliases[33]=(Command="Axis aLookUp SpeedBase=100.0 DeadZone=0.1",
Alias="SpaceFighter_JoyY")

// Rolle (=Springen+Seitenschritt) zugewiesen an zweiten Analogstick, X-Achse
Aliases[34]=(Command="Axis aUp Speed=+300.0 
| Axis aStrafe SpeedBase=300.0 DeadZone=0.1",
Alias="SpaceFighter_JoyV")

// Schub (=Nach vorne) zugewiesen an Schubkontrolle des Joysticks
Aliases[35]=(Command="Axis aBaseY SpeedBase=300.0 DeadZone=0.1 Invert=-1",
Alias="SpaceFighter_JoySlider1")

Hier nun die übliche Belegung für Joysticks (und Gampads). Bitte beachten
Sie, dass es sich hierbei lediglich um die am weitesten verbreitete
Belegung handelt und die Steuerung Ihres Controllers von dieser Belegung
abweichen kann.

JoyX = Analogstick (linker Analogstick) X-Achse
JoyY = Analogstick (linker Analogstick) Y-Achse
JoyZ = Analogstick Z-Achse, oder Schubregler (wie etwa bei Logitech(tm)
       Wingman Gamepads)
JoyR = <nicht zugewiesen>
JoyU = <nicht zugewiesen>
JoyV = Z-Rotation oder rechter Analogstick X-Achse (Ruder).
JoySlider1 = Schub (Z, Schubregler) oder rechter Analogstick X-Achse.
JoySlider2 =

Beachten Sie, dass der "Coolie-Hat" wie folgt belegt ist. Die Belegung
kann, wie bei normalen Tasten auch, direkt im Spiel geändert werden:
Joy13 = MoveForward
Joy14 = StrafeRight
Joy15 = MoveBackward
Joy16 = StrafeLeft


3.12 Logitech-Mäuse
-------------------

Haben Sie die MouseWare Software installiert und können Sie den zusätzlichen
Tasten Ihrer Logitech-Maus (mittlere Maustaste, Daumentaste, etc.) keine
Befehle zuweisen, fügen Sie folgenden Eintrag in Ihre Registry ein. 

Fügen Sie unter "HKEY_LOCAL_MACHINE -> Software -> Logitech -> MouseWare ->
CurrentVersion -> GamingCompatibility" einen neuen Eintrag namens
Unreal2004 ein und geben Sie als Wert das verzeichnis des Spiels an.
Standardmäßig wäre das "c:\UT2004\System\UT2004.EXE".
Benötigen Sie hierbei Hilfe, wenden Sie sich bitte an den Kundendienst
von Logitech (http://www.logitech.com/).

 4 Eigenständiger Server
========================================================================

4.1 Erklärung
-------------

Um bei Netzwerk-Spielen eine optimale Leistung zu garantieren, können Sie
einen eigenständigen Unreal Tournament 2004-Server auf einem Computer
starten. Ein solcher Server ist deutlich leistungsstärker als ein nicht
eigenständiger Server, allerdings wird so auch ein PC "blockiert", an dem
nicht gespielt werden kann.

Jeder hat das Recht, einen eigenständigen Server im Internet zu betreiben;
hierzu sind weder eine Erlaubnis noch eine Anmeldung erforderlich.

4.2 Starten
-----------

Sie können einen eigenständigen Server über das normale Unreal Tournament
2004 Menü starten: wählen Sie 'Mehrspieler-Spiel hosten'/'Spiel & Karte',
legen Sie die entsprechenden Einstellungen im Server-Menü fest, und
aktivieren Sie 'Eigenständiger Server'. Diese Methode ist vor allem bei
LAN-Partien sinnvoll, bei denen ein zusätzlicher PC als eigenständiger
Server zur Verfügung steht.

Alternativ können Sie einen eigenständigen Server auch über die
Befehlszeile starten, indem Sie die ut2004.exe direkt ausführen
(sie befindet sich meist im Verzeichnis c:\UT2004\System, oder aber
im Unterverzeichnis 'System' des Verzeichnisses, unter dem Sie das
Spiel installiert haben). Um zum Beispiel den Level "DM-Rankin" zu
starten, geben Sie folgendes ein:

   ucc server DM-Rankin.ut2


4.3 Mehrere Server auf einem Rechner
------------------------------------

Auf jedem eigenständigen Server kann immer nur ein Level zu einer
bestimmten Zeit laufen.

Sie können jedoch mehrere Server mit verschiedenen Leveln auf einem
Rechner starten. Dazu müssen Sie jedem Server einen bestimmten TCP/IP
Port zuweisen. Der Standard-Port ist 7777. Um einen anderen Port zu
bestimmen, geben Sie einen Befehl nach diesem Schema in die
Befehlszeile ein:

   ucc server DM-Rankin.ut2 port=7778

 5 UT2004 Statistik 
========================================================================

5.1 Übersicht
-------------

Unreal Tournament 2004 beinhaltet den frei verfügbaren UT2004stats-Service.
UT2004stats ermöglicht es Ihnen, Ihre Punkte und Position in der Rangliste
aufzuzeichnen und zu veröffentlichen. Außerdem bietet Ihnen der Service eine
statistische Analyse Ihrer Internet-Partien.

Für eine detaillierte Aufschlüsselung Ihrer Statistiken, besuchen Sie
UT2004stats unter:

	http://ut2004stats.epicgames.com


5.2 Aktivierung der Aufzeichnung *Ihrer* Statistik	
--------------------------------------------------	

Rufen Sie das Menü Einstellungen > Netzwerk > Weltw. UT2004 SPIEL STAT auf.	
Aktivieren Sie Ihre UT2004Stat., indem Sie auf die Option 'Stat. verfolgen'	
klicken und wählen Sie einen Benutzernamen sowie ein Passwort. Auf diese	
Weise erhalten Sie eine persönliche Spieler-ID .	

Hinweis: Ihre Online UT2004Stat. ist nur sicher, solange Sie Ihr Passwort	
geheimhalten.
