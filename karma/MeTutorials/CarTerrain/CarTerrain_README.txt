README - CarTerrain
~ tutorial
last updated: Dec 1, 2000


Overview:

CarTerrain is a very simple vehicle simulation with steering and 
suspension using the MdtCarWheel joint. The vehicle is controlled 
by dragging the mouse with shift in the display window. Heightfield
collision detection is used to handle the collisions between the wheels
and the uneven terrain.


Hardware requirements:

PII 300MHz, 3d acceleration
Windows98 / Win2000 / WinNT / Linux / Irix / Playstation2
