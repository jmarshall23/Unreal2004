README - Snooker
~ tutorial
Toolkits demonstrated: Collision and Dynamics
last updated: June 27, 2000


Overview:

A no-pockets European style of snooker where the objective 
is to hit combinations, not sink balls. Showing different 
contact properties of ball-ball, ball-table, and ball-cushion
collisions. 


Hardware requirements:

PII 300MHz, 3d acceleration
Windows98 / Win2000 / WinNT / Linux / Irix / Playstation2
